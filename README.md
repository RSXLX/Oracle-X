# Oracle-X NoFOMO

**AI 驱动的交易冷静层 + 钱包分析工具**

## 功能特性

### 核心功能
- 👁️ **全局监控** - 监听任意应用，检测交易操作
- 📸 **截图 AI 分析** - 识别交易按钮，实时阻断
- 💰 **钱包分析** - ETH/BSC/SOL 链上交易记录
- 📄 **CSV 导入** - 支持 7+ 交易所自动识别
- 📊 **交易习惯分析** - 风格/风险/洞察建议
- 📈 **风险评估** - 综合风险评分 + 优化建议

### 系统功能
- 🔔 **系统托盘** - 常驻后台运行
- 🖥️ **开机自启** - 登录自动启动
- ⌨️ **全局快捷键** - 快速操作
- 💾 **本地存储** - 数据隐私安全

## 支持平台

### 交易所 CSV
- Binance, OKX, Bybit, Coinbase, Kraken, Gate.io, KuCoin

### 区块链
- Ethereum, BNB Chain, Polygon, Arbitrum, Solana

## 快速开始

```bash
# 安装
cd desktop
npm install

# 启动
npm run dev
```

## 技术栈

- Electron + Node.js
- MiniMax Vision API (免费)
- 本地数据存储
